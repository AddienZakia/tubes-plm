## Tubes PLM

### Getting Started

- setup venv (pake venv biar tidak pake python secara global)

```
python -m venv venv
```

- activate venv

```
venv\Scripts\activate
```

- install requirement dependency

```
pip install -r requirements.txt
```

### Run Program

- Run Main Program

pastikan working directory pada `command prompt` sudah berada pada folder, lalu run program tersebut pada `command prompt`

```
python -m src.main
```

- Run Sandbox Program

sandbox untuk melihat contoh hasil dari component yang dibuat, pastikan working directory pada `command prompt` sudah berada pada folder, lalu run program tersebut pada `command prompt`

```
python -m src.sandbox.main
```
